# ISO 27001 GRC Project

## Overview

This project simulates the implementation of an ISO/IEC 27001:2022 Information Security Management System (ISMS) for a fictional technology consultancy, EugTech Solutions Ltd.

The project demonstrates practical Governance, Risk, and Compliance (GRC) activities including:

- Information Security Governance
- Risk Assessment and Risk Treatment
- Statement of Applicability (SoA)
- Asset Management
- Incident Response
- Business Continuity Planning
- Internal Audit
- Management Review

## Organisation

**Organisation:** EugTech Solutions Ltd (Simulated)

**Industry:** Technology Services

**Employees:** 50

**Scope:** Cloud services, managed services, and internal IT systems supporting client delivery.
